Helen Grzyb and Associates 2020 – Digital transformation

This year I was engaged as a consultant to assist with updating an existing website built on WordPress and implemented a new custom built website on a different hosting service.

Due to the tight timeframe for the project, I utilised Bootstrap 4 to quickly create a solution that would meet business needs.

The new site included a set of icons to better communicate the client's services, each icon tailored to reflect the nature of the deliverable. Bootstrap components such as inline list tabbable content have been used to make large amounts of content about the client's background, services and previous clients easier to access and digest.

I worked independently, using GIT via command line for version control and created new branches for developing each additional function. Pull requests were reviewed by another developer who assisted with reviewing my code, and subsequently merged to the master using GitHub.
